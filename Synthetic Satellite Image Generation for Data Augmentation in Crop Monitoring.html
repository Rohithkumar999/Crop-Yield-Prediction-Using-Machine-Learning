Step 2: Create Synthetic Dataset

Simulate crop features and yields to mimic real-world data

In[2]:
np.random.seed(42)  # For reproducibility
num_samples = 500

soil_moisture = np.random.uniform(0.1, 0.8, num_samples)  # Soil moisture level
temperature = np.random.uniform(10, 35, num_samples)     # Temperature in Celsius
rainfall = np.random.uniform(0, 200, num_samples)        # Rainfall in mm
ndvi = np.random.uniform(0.2, 0.9, num_samples)           # Vegetation index (NDVI)
crop_type = np.random.choice(["Wheat", "Maize", "Rice"], num_samples)  # Crop variety
season = np.random.choice(["Kharif", "Rabi"], num_samples)             # Growing season

We generate 500 data points with realistic ranges for different crop yield influencing factors.

Step 3: Generate Crop Yield with Noise

Create a target variable Yield based on features with some noise to simulate variability.

In[3]:
yield_ = (soil_moisture * 50 + temperature * 2 + rainfall * 0.3 + ndvi * 40 +
          np.where(crop_type == "Wheat", 10, 0) + np.where(crop_type == "Maize", 20, 0) +
          np.where(season == "Kharif", 5, 0) + np.random.normal(0, 5, num_samples))

Yield grows with soil moisture, temperature, rainfall, and NDVI

Additional increments based on crop type and season

Added random noise to simulate natural variation

Step 4: Create DataFrame

Combine all features and yield into a DataFrame for analysis.

In conclusion,

Machine learning models, particularly ensemble methods like Random Forest, have demonstrated strong potential for accurately predicting crop yields based on features such as soil moisture, temperature, rainfall, vegetation indices, crop type, and season. These models effectively capture complex nonlinear relationships between environmental and crop variables, enabling robust yield forecasts that can assist farmers and agricultural planners in optimizing resource allocation, managing risks, and improving decision-making.

Descriptive statistics and visualizations help reveal important trends and feature distributions, while feature importance analysis highlights which variables are most influential in predicting yields. Although challenges such as regional and seasonal variability remain, advancements in sensing technologies, remote sensing data integration, and machine learning algorithms offer promising pathways toward cost-effective, scalable, and timely yield prediction.

This approach is valuable for improving agricultural productivity and sustainability, ultimately contributing to food security in the face of climate change and growing global demand. Future work may involve integrating multi-source data, deploying automated machine learning pipelines, and adapting models for diverse geographic and crop-specific conditions to further enhance predictive accuracy and practical impact.

Overall, the use of machine learning in crop yield prediction represents a transformative step toward smarter, data-driven agriculture that supports efficient, sustainable farming practices
